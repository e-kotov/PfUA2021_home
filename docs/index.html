1.6 Типовые проблемы и решения

1.6.1 У меня русские пути к библиотеке пакетов и из-за этого не работает нормально установка пакетов

Попробуйте создать новую папку, например, c:/rpackages/ и добавьте путь к ней, следуя инструкциям по ссылке https://www.accelebrate.com/library/how-to-articles/r-rstudio-library - там см. последний раздел Making Your Library Changes Permanent
