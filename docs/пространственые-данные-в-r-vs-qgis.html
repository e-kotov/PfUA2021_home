5.2 Основные понятия

  • объект типа sf (simple features) - это особая форма data.frame, где присутствует специальным образом отформатированная колонка с геометрией для каждой строки в таблице. Документация по библиотеке sf

  • функции для загрузки пространственных данных st_read() и read_sf()

  • функции st_crs() (для проверки системы координат объекта) и st_transform() (для преобразования объекта в другую систему координат) + сайты-справочники систем координат: https://epsg.io, https://spatialreference.org

  • пространственное пересечение и выборка (spatial select / select by location): имя_объекта_1[имя_объекта_2,] позволяет выбрать все наблюдения из объекта 1, геометрия которых пересекается с наблюдениями объекта 2

  • построение буферных зон при помощи фукнции st_buffer()

  • пространственная агрегация (spatial join / join by location) с подсчетом статистик (сумма, макс, мин, среднее, и т.д.): aggregate()

  • функции для экспорта пространственных данных st_write() и write_sf()
